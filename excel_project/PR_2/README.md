**Project 2 ===> Sales Data Analysis**

This project analyzes a sales dataset containing customer, region, product category, sales, discount, and profit details to extract meaningful business insights using advanced Excel features.

The dataset includes transactions across multiple regions (Central, East, West, North, South) and product categories (Books, Clothing, Electronics, Furniture, Office Supplies), allowing comprehensive analysis of sales performance.

**Work Performed**

1. Conditional Formatting

Used to visually highlight the top 10 customers based on total purchase using color scaling and rules.

2. WHAT-IF Analysis

Performed to analyze how changes in discount impact total profit using Data Table.

3. Linear Regression (Data Analysis Toolpak)

Applied regression to understand the relationship between Sales (independent) and Profit (dependent).

4. Descriptive Statistics (Data Analysis Toolpak)

Used to summarize dataset using mean, standard deviation, min, max, and count.

5. Custom Formatting (Up/Down Arrows)

Added arrow icons to represent monthly sales growth trends (increase/decrease).

6. Timestamp Creation

Used =NOW() function to generate current date and time dynamically.

7. High-Value Customer Identification

Used INDEX + MATCH + filters to find customers with highest sales values.

8. Pivot Table Analysis

Created pivot tables to analyze total sales by region and product category.

9. Data Visualization (Charts)

Created:

Bar Chart → Sales by Region
Line Chart → Monthly Sales Trend
Pie Chart → Sales by Product Category

10. Dashboard (Data Storytelling)

Built an interactive dashboard combining KPIs, charts, slicers, and insights for decision-making.
